# Geoquiz 🌍

An interactive geography quiz game where players identify countries on a rotating 3D globe.

## Features

- Interactive 3D globe visualization using D3.js
- Random country selection for quiz questions
- Click to select countries on the globe
- Score tracking and feedback
- Smooth globe rotation and zoom

## How to Play

1. A random country name will be displayed
2. Click on the correct country on the globe
3. Get instant feedback on your selection
4. Track your score as you play

## Running the Game

### Option 1: Simple HTTP Server
```bash
npm start
```
Then open http://localhost:8000 in your browser

### Option 2: Direct File
Simply open `index.html` in a modern web browser

## Technologies

- D3.js v7 for globe visualization and projections
- TopoJSON for efficient map data
- Vanilla JavaScript for game logic
- CSS3 for styling and animations
