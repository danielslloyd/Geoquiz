# GeoQuiz

An interactive geography quiz game that tests your knowledge of countries, states, flags, and capitals.

## Game Modes

### World Quiz
Test your global geography knowledge through three sequential questions per country:
- **Location**: Click on the globe to find the country
- **Flag**: Identify the country's flag from multiple choices
- **Capital**: Select the correct capital city

Countries remain highlighted throughout all three questions - incorrect guesses stay red while correct answers show in green.

### US States Quiz
Locate all 50 US states on an interactive map, including Alaska and Hawaii inlay boxes.

### Indian States Quiz
Identify Indian states and territories on a regional map.

### Identify Countries
View a highlighted country on the globe and select its name from multiple choice options.

### Name All Countries
Type the names of as many countries as you can. Press Enter to submit each answer.

## Features

- Interactive globe with drag-to-rotate and scroll-to-zoom
- Immediate visual feedback with color-coded highlights
- Auto-advancement between question types
- Score tracking
- Compact, streamlined interface
